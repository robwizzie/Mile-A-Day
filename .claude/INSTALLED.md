| `PostToolUse` | `Write\|Edit` | `circular-deps.sh` | When 3+ files changed, runs `madge --circular` (JS/TS). Warns only — doesn't block. **`madge` not installed locally — install via `npm i -g madge` to activate.** |
| `PostToolUse` | `*` | `token-telemetry.sh` | Logs estimated token spend to `.claude/telemetry/spend.jsonl`. Falls back to char-count if `tiktoken` not available. |
